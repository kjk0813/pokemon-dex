# Pokemon Dex - プロジェクト引き継ぎ

## プロジェクト概要

- **構成**: `index.html` 1ファイルのモノリシック構成（バニラJS + Canvas、フレームワークなし）
- **既存機能**: ポケモン捕獲ゲーム・図鑑・ビレッジ（畑あり）・ミニゲーム群（7種以上）
- **対応**: スマホファースト、localStorage でデータ永続化、PWA対応

---

## 実装予定：育てる要素（なつき度 × きのみ畑）

### 全体の流れ

```
畑でキノミを育てる → 収穫 → ポケモンに与える → なつき度UP → 好みを聞き出す → 親友へ
```

### なつき度フェーズ

| フェーズ | なつき度 | できること |
|---------|---------|-----------|
| 知らない仲 | 0〜29 | きのみを与える（+10） |
| 友だち | 30〜59 | 「好みを聞く」ボタンが解放 |
| 仲良し | 60〜89 | 好みのきのみで +30 |
| 親友 | 90〜100 | 親友バッジ・特別セリフ |

### きのみの種類

| きのみ | なつき度UP |
|--------|-----------|
| オレンのみ | +10 |
| ラムのみ | +15 |
| ヤチェのみ | +20 |
| 好みのきのみ（ポケモンごと） | **+30** |

### 好みのきのみ「聞き出し」仕様

- 各ポケモンに隠し属性として好みのきのみが設定されている
- なつき度30未満 → 好みは「🔒 ???」表示
- なつき度30以上 → 「好みを聞いてみる」ボタンが出現
- ボタンを押すと吹き出しで教えてくれる（例：「…ラムのみ、かな」）
- 判明後は図鑑カードに「好きなきのみ: 🍋 ラムのみ ✨」と表示

### データ構造（localStorage）

```js
// ポケモンごとの好みきのみ（隠し定数）
const favoriteBerry = {
  133: "ラムのみ",   // イーブイ
  134: "オレンのみ", // シャワーズ
  135: "ヤチェのみ", // サンダース
  // 他のポケモンも同様に定義
}

// localStorage に保存するプレイヤー状態
{
  friendship: { 133: 45, 134: 12 },      // なつき度
  berryKnown: { 133: true, 134: false },  // 好みを聞き出したか
  bag: { "オレンのみ": 3, "ラムのみ": 1 } // 所持きのみ
}
```

## ビジュアル方針（どうぶつの森風）

- **見下ろし2.5D**：楕円グラデの地面プレート
- **ポケモンはぷかぷか浮く**：`translateY` アニメーション + 楕円影が拡縮
- **きのみを与えた瞬間**：ハートパーティクルが飛ぶ演出
- **吹き出し**：丸みのある白いふきだしでセリフ表示
- **カラー**：パステル系（空 `#87CEEB`・草 `#7EC850`・パネル `#FFFDF0`）

ビジュアルサンプルは `nurture-sample.html` を参照（ブラウザで開くと動作確認できる）。

## 実装ステップ（推奨順）

1. `localStorage` のデータ構造追加（friendship・berryKnown・bag）
2. 畑にきのみ3種を追加（既存の作物コードを拡張）
3. 図鑑カードにUI追加（ゲージ・ボタン・好みきのみ表示）
4. なつき度の加算ロジック実装（きのみ・話しかける・ミニゲーム）
5. 「好みを聞く」インタラクション実装
